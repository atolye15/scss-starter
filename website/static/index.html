Because We Open Source

We have been using SCSS starter kit internally for a while now and we have been developing it based on our needs, and our needs vary! But of course we didn’t compromise it by adding unrelated things.

SCSS Starter Kit works as a handy toolbox that many web developer could get benefit from. We took care of the stress of creating semantic folder structure so that our web developers focus on the real developing!
